This application is some sort of a clone of imdb website. At least CSS in nav bar. 
You can use main search for movies or switch between three categories - movies, TV shows or celebrities and add some of them to your favourest.  The list of your favourite celebrities you can sort by alphabet or use search through that list.
Movie description is added to the backside of movie cards. For celebrities it is a list of some their movies.

This app is in folder 2021 on https://alexthunders.ru
